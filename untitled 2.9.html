Un poco más de HTML

El elemento <br> se utiliza para insertar un salto de línea dentro de un texto. Esto permite que el contenido continúe en la siguiente línea sin necesidad de crear un nuevo párrafo.

Ejemplo de uso de <br>:
Este es un texto.

Este es otro texto en la siguiente línea.
